# MonPortFolio – Personal Portfolio Website

A personal portfolio website presenting projects, skills, and contact information. This project serves as a professional showcase of technical and creative work for recruiters, clients, and collaborators.

---

## Project Purpose

This website displays key information about the author’s background, skills, and completed projects in a clean, responsive layout — ideal for CV presentation, professional visibility, and portfolio storytelling.

---

## Prerequisites

Before running locally:

- A modern web browser (Chrome, Firefox, Edge, Safari)
- (Optional) A local HTTP server for development

---

## Local Deployment

### 1. Clone the Repository

```bash
git clone https://github.com/EmmaDjeufa/MonPortFolio.git
cd MonPortFolio
````

### 2. Open in Browser

You can open the main file directly:

```bash
open index.html
```

Or serve it locally (optional):

```bash
# Using Python
python3 -m http.server 8000
```

Then go to:

```
http://localhost:8000
```
